Shellscript Basics:
	The shell is a Command Line Interpreter. It translates commands entered by the user and converts them into the language that is understood by the kernel. This use the standard syntax for all the commands.
	The basic concept of a shell script is a list of commands, which are listed in the order of execution. A good shell script will have comments, preceded by (#)sign, this describes the steps. [it is not mandatory to use the "#" sign, but it is a good practise, if someone else going to go through the script that makes easy for them to understand the script easily]

What is Variable?
	A variable is a character string to which we assign a value. The value assigned could be a number, text, filename, device or any other type of data. [in other-words, variable is a pointer to the actual data]

Kinds of Variable inside a shell:
1. Local Variable: It is a variable that is present within the current instance of the shell. It is not available to programs that are started by the shell. 
2. Environment Varibale: It is available to any child process of the shell, some programs need environmant variables in order to function correctly.
3. Shell Variable: It can be comapred/considered to be the global variable that is set by the shell and is required by the shell in order to function correctly. Some of this variables are environment variables whereas others are local variables. It is also considered to be the special variable.

Shell loops:
	This is considered to be the very powerful programming tools that enables you to execute a set of commands repeatedly. Some of the loops are like (The While Loop, The For Loop, The Until Loop, Nested Loops, Loop Control etc.,)

The For Loop:
  This one operates on the list of items, it repeats a set of commands for every item in the list.

create a file (vi for.sh)

#!/bin/sh

for var in 0 1 2 3 4 5 6 7
do
echo $var
done

now, give the permission (chmod 777 for.sh) - and execute it (./for.sh), u can see the output got listed.

The While Loop:
  This enables to executes set of commands repeatedly until some condition occurs

create a file (vi while.sh)

#!/bin/bash

x=1
while [ $x -le 5 ]
do
  echo "Welcome $x times"
  x=$(( $x + 1 ))
done

now, give the permission (chmod 777 while.sh) - and execute it (./while.sh), u can see the output for 5 times as per instructed.

The Until Loop:
  We need to execute a set of commands until a condition is true, that is where we use this loop

create a file (vi until.sh)

#!/bin/sh

a=0

until [ ! $a -lt 10 ]
do
   echo $a
   a=`expr $a + 1`
done

here, until is not less than 0, it is going to keep on printing the a and incrementing the value, now, give the permission (chmod 777 until.sh) - and execute it (./until.sh)

Nested Loops:
   All the concepts supports nesting concepts; where you can put one loop inside another similar or different loop. This can go upto unlimited number of times based on what you required. Let's use the nesting by bring while loop

create a file (vi nloop.sh)

#!/bin/bash
# A shell script to print each number five times.
for (( i = 1; i <= 5; i++ ))      ### Outer for loop ###
do

    for (( j = 1 ; j <= 5; j++ )) ### Inner for loop ###
    do
          echo -n "$i "
    done

  echo "" #### print the new line ###
done 

now, give the permission (chmod 777 nloop.sh) - and execute it (./nloop.sh)

Infinite Loop:
   This stands for the loop going on and on

create a file (vi infi.sh)

#!/bin/bash
while :
do
	echo "Press [CTRL+C] to stop.."
	sleep 1
done

now, give the permission (chmod 777 infi.sh) - and execute it (./infi.sh), as you can see it is keep on running until we are stopping from our end.
