what is deployment?
	It is a process of Installing/Updating a code on top of the real-time/runtime server
both installation and update are considered to be deployment; ex: applications we are using day by day in google play store

webserver - static application  this will be constant one; if there is any changes need to be updated it will be done by Administrator		ex: Apache, IHS, IIS
Apache is a free-tier product
IHS - (IBM HTTP Server) it is an IBM product; for this we need to get license tool
IIS - Internet Information service - microsoft product; so we can do this only on windows machine

appserver - dynamic application 	(weighted application) which will get changed regularly	ex: WAS, weblogic, JBOSS
WAS - Websphere application server - IBM product
weblogic - oracle product
JBOSS - open-source product
All this applications are used to maintain heavy weighted applications like FB, twitter etc.,

This will be decided by developer according to the client need
